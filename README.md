# vantum-backend
Backend vantum (API to apply for Jr Web Developer)


## Getting Started/Installation
To run: 
```
npm run dev
```
To test:
```
npm run test
```

### Prerequisites
```
NodeJS
```

## Built With

* [NodeJS](https://nodejs.org)
* [Express](https://expressjs.com) 
* [Mocha/Chai](https://mochajs.org/) - Used to run some tests
